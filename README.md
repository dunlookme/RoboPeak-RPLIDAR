RoboPeak-RPLIDAR
================

This project has been created to research how to use RPLIDAR with Arduino.

![ScreenShot](http://rplidar.robopeak.com/img/demoimage.gif)

## Connections

RPILAR has 7 pins:

    GND
    RX
    TX
    5V
     
    GND
    MOTOCTL
    VMOTO
